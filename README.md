## 🍱 What's Inside the Box?

This portfolio is a lightweight, static frontend built without heavy frameworks. It focuses on clean CSS grid architecture and interactive data storytelling. 

* **The Bento Grid:** A modular, tactile layout that organizes my TPM experience, MS Data Science coursework, and sustainability consulting into easily scannable cards.

## 🛠️ Tech Stack

* **Frontend:** HTML5, modern CSS3 (CSS Grid/Flexbox), Vanilla JavaScript.
* **Data & Analytics:** Python, Pandas, GeoPandas, NumPy, Folium (for the geospatial PACE model).
* **Hosting:** GitHub Pages.
